# 🩺 Medical Diagnosis System

A **Symptom-Based Disease Prediction System** built using Java that allows users to select symptoms and get predicted diseases along with suggested treatments.

---

## 🚀 Features

* Select symptoms through an interactive GUI
* Predict diseases based on selected symptoms
* Display recommended treatments
* MySQL database integration
* Optional API-based prediction support

---

## 🛠️ Tech Stack

* **Java (Swing, JDBC)**
* **MySQL**
* **REST API (optional)**

## ⚙️ Setup

1. Clone the repo
   git clone https://github.com/your-username/medical-diagnosis-system.git

2. Create database
   CREATE DATABASE medical_diagnosis_db;

3. Update DB credentials in code

4. Run `Main.java`
